# Yahoo
Projet : Yahho (Powered by Bootstrap)
​Description :
​"Une interface web moderne et entièrement responsive réalisée avec le framework Bootstrap 5. Ce projet met en avant l'utilisation des composants prédéfinis, du système de grille (Grid System) et des classes utilitaires pour créer une expérience utilisateur fluide et rapide sur tous les types d'écrans (Mobile, Tablette, Desktop)."
​Technologies utilisées :
​Bootstrap 5
​HTML5 / CSS3
​Responsive Design.
